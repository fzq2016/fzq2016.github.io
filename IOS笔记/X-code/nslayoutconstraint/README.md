## 适配
- 适应、兼容各种不同的情况
- 移动开发中适配的常见种类
    - 系统适配
        - 针对不同版本的操作系统进行适配
    - 屏幕适配
        - 针对不同大小的屏幕尺寸进行适配
        - iPhone的尺寸
            - 3.5inch(320*480)   IPhone4及4s及之前版本
            - 4.0inch(320*568)   IPhone5及5s、SE
            - 4.7inch(375*667)   IPhone6及6s、7
            - 5.5inch(414*736)   IPhone6Plus、7Plus
        - iPad的尺寸
            - 7.9inch   IPadMini
            - 9.7inch   IPad
        - 屏幕方向
            - 竖屏
            - 横屏
