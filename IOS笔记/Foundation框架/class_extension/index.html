类扩展(Class Extension)

  1. 作用
    • 能为某个类增加额外的属性、成员变量、方法声明
    • 一般将类扩展写到.m文件中
    • 一般将一些私有的属性写到类扩展
  2. 使用格式
      @interface 类名()
  /* 属性、成员变量、方法声明 */
  @end

  3. 与分类的区别

    1. 分类的小括号必须有名字

      @interface 类名(分类名字)
/* 方法声明 */
@end

@implementation 类名(分类名字)
/* 方法实现 */
@end
    2. 分类只能扩充方法
    3. 如果在分类中声明了一个属性，分类只会生成这个属性的get\set方法声明
