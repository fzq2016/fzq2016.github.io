UIKit-View

UILabel（标签）

  1. 父类是UIView
  2. UILabel极其常用，功能比较专一：显示文字
  3. UILabel显示三要素：设置frame，text，加入视图
  4. 要注意UILabel如果不设置frame,还是会有一个小黑点出现在屏幕中的

UILabel常用属性

// 设置文本文字
label.text = @"label";

// 设置字体大小
// 按钮、tableViewCell可以通过获取子控件label设置字体大小
label.font = [UIFont systemFontOfSize:18];

// 设置文本颜色
label.textColor = [UIColor whiteColor];

// 设置文本对齐模式（比如左对齐、居中对齐、右对齐）
label.textAlignment = NSTextAlignmentCenter;

// 设文本行数,0表示自动换行
label.numberOfLines = 0;

// 设置文本换行模式
// line: "abcd..."
label.lineBreakMode= NSLineBreakByTruncatingTail;

计算UILabel中文字的宽度

CGFloat textW = [字符串 sizeWithFont:字体大小].width;
CGFloat textW = [字符串 sizeWithAttributes:@{NSFontAttributeName : 字体大小}].width;

UILabel实现包裹效果（UILabel随内容变化）

  1. storyboard
    • 设置宽度约束为 <= 固定值
    • 设置位置约束
    • 不用设置高度约束

  2. 代码实现

      // 设置label自动换行(不超过1行可不设置)
  label.numberOfLines = 0;

  // 生成字体大小属性字典
  NSDictionary *textAttributes = @{NSFontAttributeName : [UIFont systemFontOfSize:14]};

  // 设置最大尺寸
  CGFloat labelW = [UIScreen mainScreen].bounds.size.width - 20
  CGSize textMaxSize = CGSizeMake(labelW, MAXFLOAT);

  // NSStringDrawingUsesLineFragmentOrigin指文字从开头位置往后排布
  // 下述代码实现功能：假设字符串中每个文字的大小是14号字体，将字符串从开头往后排布，当文字即将超出每行宽度限制时，自动换行，计算出合理的size
  CGFloat labelH = [self.text boundingRectWithSize:TEXTMaxSize options:NSStringDrawingUsesLineFragmentOrigin attributes:TEXTAttributes context:nil].size.height;

  // label是自动换行的，只要给出合适的label尺寸，label中的文字就不会被挡住而无法显示,这样就能根据文字内容自动调整label尺寸
  label.frame = CGRectMake(0, 0, labelW, labelH);

UILabel实现指示器效果

  • 半透明的指示器如何实现？

    • 设置指示器的背景色是半透明的

    • 动画实现指示器的alpha 从 1.0 -> 0

      // 半透明背景色
self.hudLbl.backgroundColor = [UIColor colorWithRed:1 green:1 blue:0 alpha:0.5];

// 在1秒内执行动画
  [UIView animateWithDuration:1.0 animations:^{
      // 显示指示版
      self.hudLbl.alpha = 1;//不透明

  } completion:^(BOOL finished) {

      // 上一个动画完毕后延迟3秒后，在1秒内执行下列动画
      [UIView animateWithDuration:1.0 delay:3.0 options:kNilOptions animations:^{

          // 隐藏指示版
          self.hudLbl.alpha = 0;//完全透明
      } completion:nil];
  }];
