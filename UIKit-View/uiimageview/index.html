UIKit-View

UIImageView

  • 父类是UIView
  • UIImageView极其常用，功能比较专一：显示图片

UIImageView的常见属性

// 显示的图片
imageView.image = image;

// 加载动画图片集
imageView.animationImages = imageArr;

// animationDuration：动画图片的持续时间
// 1秒播放30张图片
imageView.animationDuration = imageCount  * 1 / 30.0;

// 动画的播放次数（默认是0，代表无限播放）
imageView.animationRepeatCount = 0；

UIImageView的常见方法

  1. initWithImage：

     利用initWithImage创建出来的imageView的尺寸和传入的图片尺寸一样

     UIImageView *imageView = [[UIImageView alloc] initWithImage:image];

  2. startAnimating：开始动画

      [imageView startAnimating];

  3. stopAnimating：停止动画

    • 使用场景：根据播放时间设置停止动画
      [imageView stopAnimating];

  4. isAnimating：是否正在执行动画

      BOOL isAnimating = imageView.isAnimating;
