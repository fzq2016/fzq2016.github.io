UIBarItem

  • 父类是NSObject

UIBarItem常用属性

// 设置是否可用，默认为YES
batItem.enabled = YES;

// 设置模型文字属性，默认为nil
batItem.title = @"UIBarItem";

// 设置模型图片属性
batItem.image = [UIImage imageNamed:imageName];

模型与控件

  1. 按照封装的思路，一般将所有控件封装起来，只向外部提供模型属性，外界修改模型的属性后，封装的控件内部会自动设置控件数据
  2. 苹果自然也按该思路封装，因此只要看到item,通常都是苹果提供的模型,只要改模型就能修改苹果的某些控件.
  3. 导航控制器的导航条、标签控制器的标签条、地图的大头针控件都遵循这种思路。要修改控件的内容，就修改模型属性
